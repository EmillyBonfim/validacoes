### CPF

| Elementos | Atributos |
| --- | --- |
| ``<*form*>`` - formulário | *input* - Dados requisitados
| ``<*label*>`` - Rótulo | *type* - Tipos de dados(password, email e submit)
| ``<*button*>`` - Botão | *id* - Indentificador
| ``<*div*>`` - Agrupa os conteúdos | *placeholder* - Legenda os campos de dados
